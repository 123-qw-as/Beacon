# 优化模型

## 适用场景

在约束条件下求目标最优（最大/最小）。典型竞赛题：调度（车辆/人员/设备排班）、
覆盖（无人机/基站选址）、路径规划（物流配送）、资源分配（预算/产能）。
判别特征：有"最小化成本""最大化覆盖""不超过""最优方案"等关键词。
按变量类型分：线性规划（LP）、整数规划（IP/MIP）、非线性规划（NLP）。

## 核心公式

线性规划标准形：

    min  c^T x
    s.t. A x ≤ b,  x ≥ 0

整数规划：x ∈ ℤ^n（0-1 规划是 x ∈ {0,1}^n）。
非线性规划：目标或约束含 x 的非线性项（x^2、exp(x)、x·y 等）。

运输问题（LP 特例）：m 个供应 a_i，n 个需求 b_j，运价 c_ij，
    min Σ c_ij x_ij  s.t. Σ_j x_ij = a_i, Σ_i x_ij = b_j, x_ij ≥ 0

指派问题（0-1 IP）：n 人 n 任务，效率矩阵 c_ij，
    min Σ c_ij x_ij  s.t. Σ_j x_ij = 1, Σ_i x_ij = 1, x_ij ∈ {0,1}

## 建模步骤

1. 定变量：把决策对象抽象为 x = (x1,...,xn)，注明单位与取值域（连续/整数/0-1）。
2. 定目标：写出 min/max 的目标函数，确认是线性还是非线性。
3. 定约束：把每个"不超过/至少/恰好"翻译为不等式/等式，含非负约束。
4. 求解：LP 用单纯形法（scipy.optimize.linprog）；MIP 用分支定界（PuLP/SCIP）；
   NLP 用 SLSQP/ipopt。规模大时考虑启发式（遗传/模拟退火）。
5. 灵敏度：对关键参数（成本/容量）做 ±10%~50% 扰动，看最优解/目标值变化。

## 常见陷阱

- 变量连续 vs 整数没说清：调度"派 3.7 辆车"无意义，应是整数规划。
- 约束遗漏非负：x ≥ 0 是隐含约束，linprog 不加 bounds 会得到负解。
- 多目标硬加权：题目要"低成本且高服务质量"，直接加权 c1·成本 - c2·质量，
  量纲不同导致权重无意义。应先归一化，或用 Pareto 前沿 / 目标规划。
- NLP 初值敏感：SLSQP 对初值敏感，多试几个起点避免局部最优。
- 忘记验证可行性：求解器返回"不可行"时，检查约束是否矛盾（如总供给 < 总需求）。
