# 微分方程模型

## 适用场景

随时间连续演化的过程。典型竞赛题：种群增长（Logistic）、传染病（SIR/SEIR）、
物理过程（热传导/扩散）、化学反应动力学、交通流。判别特征：有"变化率""随时间""演化"
"增长/衰减"等关键词，且过程是连续的（非离散步进）。

## 核心公式

### Logistic 增长（单种群）
dx/dt = r x (1 - x/K)，x(0)=x0。
- r：内禀增长率；K：环境容纳量。
- 解：x(t) = K / (1 + (K/x0 - 1) e^{-rt})，S 形曲线趋近 K。

### SIR 传染病模型
三类人群：S 易感、I 感染、R 恢复（免疫）。
    dS/dt = -β S I / N
    dI/dt = β S I / N - γ I
    dR/dt = γ I
- β：感染率；γ：恢复率；基本再生数 R0 = β/γ。
- R0 > 1 爆发，R0 < 1 自行消退。群体免疫阈值 1 - 1/R0。

### SEIR（加潜伏期 E）
    dS/dt = -β S I / N
    dE/dt = β S I / N - σ E        # σ：潜伏期转阳率
    dI/dt = σ E - γ I
    dR/dt = γ I

## 建模步骤

1. 定状态变量与守恒关系：人群总数 N = S+I+R 是否守恒（有出生死亡则不守恒）。
2. 写速率方程：每个状态的流入/流出项，确认单位（人/天 vs 人/小时）。
3. 定参数：β/γ/r 用真实数据拟合（最小二乘 / 贝叶斯）或文献取值，注明来源。
4. 求解：解析解（少数可积，如 Logistic）或数值积分（scipy.integrate.odeint / solve_ivp）。
5. 稳定性分析：求平衡点，Jacobian 特征值判断稳定性；R0 计算。

## 常见陷阱

- 量纲不一致：β 单位是"1/(人·天)"，γ 单位是"1/天"，R0=β/γ 才无量纲。
- 忘记守恒检验：数值积分后 Σ(状态) 应 ≈ N，偏差大说明数值误差或方程写错。
- 刚性方程：σ/γ 差异大时系统刚性，odeint（lsoda）能处理但 RK45 会卡，换求解器。
- R0 算错：R0 = β S0 / (γ N)，不是 β/γ（那是 S0=N 时）。初始易感占比影响实际 R0。
- 稳定性只看平衡点不看暂态：R0<1 但初始 I 很大时仍会短期高峰，须画 I(t) 曲线。
