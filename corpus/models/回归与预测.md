# 回归与预测模型

## 适用场景

从历史数据预测未来值或拟合变量关系。典型竞赛题：需求预测（单车/电力/销量）、
因素分析（某因素对结果的影响程度）、趋势外推。判别特征：有"预测""拟合""影响因素"
"未来 N 期"等关键词，且有历史数据表。常用方法：线性回归、灰色预测 GM(1,1)、时间序列。

## 核心公式

### 线性回归
y = β0 + β1 x1 + ... + βp xp + ε
- 最小二乘：β̂ = (X^T X)^{-1} X^T y
- R² = 1 - SS_res/SS_tot，越接近 1 拟合越好
- 调整 R² = 1 - (1-R²)(n-1)/(n-p-1)，惩罚自变量个数
- F 检验整体显著性，t 检验单系数显著性（H0: βj=0）

### 灰色预测 GM(1,1)（小样本）
原始序列 x^(0) = (x1,...,xn)，n 小（≥4 即可）。
1. 累加生成 x^(1)_k = Σ_{i=1}^k x^(0)_i
2. 白化方程：dx^(1)/dt + a x^(1) = b
3. 解 x̂^(1)_{k+1} = (x1 - b/a) e^{-ak} + b/a，累减还原 x̂^(0) = x̂^(1) - x̂^(1)_{k-1}
4. 后验差比 C = S2/S1（S1 原始序列标准差，S2 残差标准差），C<0.35 优，<0.5 合格

### 时间序列 ARIMA(p,d,q)
- d：差分阶数（使序列平稳）
- AR(p)：x_t = φ1 x_{t-1} + ... + φp x_{t-p} + ε_t
- MA(q)：x_t = ε_t + θ1 ε_{t-1} + ... + θq ε_{t-q}
- 定阶：ACF/PACF 截尾判断 p/q，ADF 检验判断 d

## 建模步骤

1. 数据探索：画时序图看趋势/季节性/异常点；算自相关判断平稳性。
2. 选模型：线性关系→线性回归；小样本无规律→灰色预测；有自相关→ARIMA。
3. 拟合与诊断：回归看残差是否白噪声（DW 检验），ARIMA 看 Ljung-Box 检验残差。
4. 验证：留出末尾 20% 数据做 out-of-sample，比较 MAPE/RMSE；不要只看训练集 R²。
5. 预测：给出点预测 + 区间预测（回归用置信区间，ARIMA 用 forecast interval）。

## 常见陷阱

- 多重共线性：自变量高度相关导致 β 方差爆炸，方差膨胀因子 VIF > 10 须剔除或岭回归。
- 过拟合：加多项式项把 R² 刷到 0.99 但外推发散，用调整 R² 和交叉验证抑制。
- 灰色预测滥用：GM(1,1) 只适合单调增长序列，波动或下降序列会预测失效。
- ARIMA 不平稳就拟合：非平稳序列直接 ARIMA 参数估计无意义，必须先差分到平稳。
- 只报点预测不报区间：评委要看预测的不确定性，纯点预测可信度低。
- 异常点不处理：单个异常值能严重扭曲回归系数，先箱线图识别再决定剔除或稳健回归。
