LangGraph Pipeline

建模流水线

  1. Analyst题意拆解与 Problem Blueprint
  2. Blueprint Critic审查题目理解完整性
  3. Modeler建立基础与改进模型
  4. Model Critic检查假设与推导漏洞
  5. Coder生成可运行实验代码
  6. Code Consistency模型-代码一致性审查
  7. Sensitivity参数扫描与鲁棒性分析
  8. Figure Pipeline图表渲染与质量评审
  9. Writer分章节生成论文草稿
  10. Paper Critic论文质量与格式评审
  11. Table Assembler注入表格并清洗正文
  12. Evaluation综合评分与改进建议
  13. Human Review等待人工批准或拒绝
  14. LaTeX编译 PDF 并回退 Markdown
